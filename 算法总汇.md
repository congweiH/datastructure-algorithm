# 3 数学

## 3.1 整除分块 $\sum_{i=1}^{n}\lfloor \frac{n}{i} \rfloor$ 

计算整除求和


$$
\sum_{i=1}^{n}\lfloor \frac{n}{i} \rfloor
$$
发现 n/i 很多值都是相同的，并且呈块状，相同的值在一起，并呈递减形式。因此我们可以将 n/i 相同的值作为一个块一起计算，通过打表发现最后一个数的位置是 n/(n/i)，这就知道了每一个块结束的位置。就可以 $\sqrt{n}$ 解决这个问题了。

```c++
ll solution(ll n){
    int sum = 0;
    for(int l=1,r; l<=n; l=r+1){
        r = n/(n/l);
        sum += (r-l+1)*(n/l);
    }
    return sum;
}
```


## 3.2 1到n所有数约数个数和

```c++
ll solution(ll n){
    ll sum = 0;
    for(int i = 1; i <= n; i++){
        sum += n/i;
    }
    return sum;
}
// sum(3) = 5
// 1有1个约数:1
// 2有2个约数:1,2
// 3有2个约数:1,3
// n/i表示i这个约数有多少个, 当i=1,有3个;当i=2,有1个; 当i=3,有1个
```

## 3.3 1到n所有数的约数和 

```c++
ll solution(ll n){
    ll sum = 0;
    for(int i = 1; i <= n; i++){
        sum += (n/i)*i;		// n/i表示i这个约数有多少个
    }
    return sum;
}
```

考虑用整除分块降低复杂度

```c++
ll solution(ll n){
    ll sum = 0;
    for(int l=1,r; l<=n; l=r+1){
        r = n/(n/l);
        sum += (r-l+1)*(n/l)*(l+r)/2;		// (l+r)/2 是这个区间i的平均值
    }
    return sum;
}
```

## 3.4 快速幂

```c++
// 求 a^b
ll qpow(ll a, ll b){
    ll res = 1;
    for(;b;b>>=1){
        if(b&1) res *= a;
        a *= a;
    }
    return res;
}
```
